Picture scraper
================

Scrap's different picture source.

#### Supported sources

* **Facebook** - Scrap facebook profile's and downloads them

#### Install

1. Clone repository
2. `npm install` in repository dir
3. node app.js
4. (opt) STRG-C to abort scraping process
